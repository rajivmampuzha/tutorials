Configure Atom editor
---------------------
Atom plugins are below.
1. Install emmet :- Html editor helper
2. atom-beautify
3. autocomplete-paths
4. api-docs
5. linter (This checks the style and syntax of code).This is base package. Language specific linters has to installed.
6. linter-csslint
7. linter-htmlhint
8. linter-eslint

Reference
---------
https://devdocs.io this site will download web tools documentations from Mozilla Developer Network (MDN).


Install Browser sync module to node
-----
npm install -g browser-sync
